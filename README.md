# Zaney Tarot MVP

A static tarot reading web app with generated anime, pixel, manga, symbolic, and pendant-style card decks.

## Local Preview

```bash
npm run dev
```

Then open `http://127.0.0.1:4173/`.

## Vercel

This is a static site. Import the GitHub repository into Vercel and keep the default settings:

- Framework preset: Other
- Build command: none
- Output directory: `.`
